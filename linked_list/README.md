# Linked List

Linked list is a type of linear data structures.

Lookup - O(N)
Insertion - O(1)
Deletion - O(1)

Application scenarios:
1. You need to do insertion and deletion a lot.
2. You don't know how many nodes you will have, for example block chain (Bitcoin).
